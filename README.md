This program implements insertion sort using C programming language.
-It prompts a user to enter the number of elements
-Accepts integer inputs as per the number of elements entered
-Sorts the array using Insertion Sort 
-Displays the sorted array.

It has following time complexities:
1.Best Case: O(n)
Occurs when the array is already sorted.
2.Average Case: O(n²)
Occurs when the elements are in random order.
3.Worst Case: O(n²)
Occurs when the array is sorted in reverse order.
4.Space ComplexityO(1)
